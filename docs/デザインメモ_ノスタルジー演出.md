## パララックス窓の強化

現状: KV画像がそのまま表示されている。

| 案 | CSS | 効果 |
|----|-----|------|
| **セピア化 + 軽いブラー** | `filter: sepia(0.6) saturate(0.7) blur(1px)` | 「あの頃」の記憶が窓から覗く |
| **モノクロ** | `filter: grayscale(1) contrast(0.9)` | 古い写真感 |
| **窓ごとに時代を変える** | 上の窓ほど `grayscale` 強め、下に行くほどカラーに近づく | 時系列で記憶が鮮明になる |
| **スクロール連動でセピア→カラー** | JS で `sepia()` の値をスクロール量に応じて変化 | 記憶が蘇る演出 |

おすすめ: **窓ごとに時代を変える** + **セピア化**

```css
/* 例: 1つ目の窓（古い記憶） */
.parallax-kv-1 { filter: sepia(0.7) saturate(0.5) blur(1.5px); }

/* 2つ目の窓（少し鮮明に） */
.parallax-kv-2 { filter: sepia(0.3) saturate(0.8); }

/* 3つ目の窓（ほぼカラー） */
.parallax-kv-3 { filter: sepia(0.1) saturate(0.95); }
```

## CSS のみで実装できる演出

### フィルムグレイン
8mmフィルム・古い映像の質感。現在 Hero に軽く適用済み、パララックス窓にも追加可能。

```css
.film-grain::after {
  content: '';
  position: absolute;
  inset: 0;
  opacity: 0.04;
  background-image: url("data:image/svg+xml,...feTurbulence...");
  pointer-events: none;
}
```

### ビネット（周辺減光）
カメラのファインダー越しの雰囲気。パララックス窓の `::after` に適用。

```css
.vignette-window::after {
  background: radial-gradient(ellipse at center, transparent 40%, rgba(0,0,0,0.4) 100%);
}
```

### 色褪せグラデーション
セクション端に向かって彩度が落ちる。記憶がぼやける感覚。

```css
.fade-edges::before {
  background: linear-gradient(
    to bottom,
    rgba(250,247,242,0.8) 0%,
    transparent 15%,
    transparent 85%,
    rgba(250,247,242,0.8) 100%
  );
}
```

### 紙のテクスチャ
古い便箋・新聞の質感。手紙セクションに追加するとリアリティが増す。

```css
.paper-texture {
  background-image:
    url("data:image/svg+xml,...feTurbulence baseFrequency='0.7'..."),
    linear-gradient(to bottom, #faf7f2, #f5efe5);
  background-blend-mode: multiply;
}
```

## JS で実装する演出

### スクロール連動ぼかし→フォーカス
思い出が鮮明になる瞬間。セクションが視界に入ると `blur(4px)` → `blur(0)` に変化。

```js
const blurObserver = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    const ratio = entry.intersectionRatio;
    const blur = (1 - ratio) * 4;
    entry.target.style.filter = `blur(${blur}px)`;
  });
}, { threshold: Array.from({ length: 20 }, (_, i) => i / 20) });
```

### スクロール連動セピア→カラー
パララックス窓の `filter` をスクロール位置で動的に変更。

```js
window.addEventListener('scroll', () => {
  const scrollRatio = window.scrollY / document.body.scrollHeight;
  const sepiaValue = Math.max(0, 0.7 - scrollRatio);
  parallaxElements.forEach(el => {
    el.style.filter = `sepia(${sepiaValue}) saturate(${0.5 + scrollRatio * 0.5})`;
  });
});
```

### タイプライター表示
手紙セクションで1文字ずつ表示。手紙を読んでいる感覚。
ただしパフォーマンス・アクセシビリティに注意（`prefers-reduced-motion` で無効化する）。

```js
function typewriter(el, speed = 30) {
  const text = el.textContent;
  el.textContent = '';
  let i = 0;
  const timer = setInterval(() => {
    el.textContent += text[i++];
    if (i >= text.length) clearInterval(timer);
  }, speed);
}
```

## 優先度の推奨

1. **パララックス窓のセピア化 + ビネット** — 最小工数で最大効果
2. **カードサイズ調整** — 視認性改善
3. **窓ごとの時代変化（grayscale 段階）** — ストーリー性が出る
4. **ぼかし→フォーカス** — エモさ増し。ただしモバイル性能に注意
