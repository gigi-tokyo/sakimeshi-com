# さきめし (sakimeshi.com) ダッシュボードサイト構築計画

## 概要

コロナ禍で飲食店を支援した「さきめし」サービスの跡地として、GitHub Pages 上にダッシュボードサイトを構築する。さきめしの社会的成果を数値で可視化し、メディア掲載実績・受賞歴を一覧表示し、Gigi が社会に果たした貢献を恒久的に記録するサイトとする。

## Part 2: ダッシュボード数値の提案

### メイン KPI（ヒーローセクション — 大きく4つ表示）

| # | 指標名 | 表示例 | DB クエリイメージ |
|---|--------|--------|-----------------|
| 1 | **総支援金額** | ¥XXX,XXX,XXX | 決済テーブルの SUM(amount) |
| 2 | **支援された店舗数** | X,XXX 店舗 | 支援を受けた店舗の DISTINCT COUNT |
| 3 | **支援者数（延べ）** | XX,XXX 人 | 購入トランザクションの COUNT or DISTINCT user |
| 4 | **支援回数** | XX,XXX 回 | 購入トランザクションの COUNT |

### サービス規模指標

| # | 指標名 | DB クエリイメージ |
|---|--------|-----------------|
| 5 | 登録店舗数（累計） | 店舗テーブルの COUNT |
| 6 | 展開自治体/パートナー数 | 固定値（上記リストから） |
| 7 | サービス運営期間 | 2020/03 〜 最終取引日（XX ヶ月間） |
| 8 | 登録ユーザー数（累計） | ユーザーテーブルの COUNT |

### ピーク・ハイライト指標

| # | 指標名 | DB クエリイメージ |
|---|--------|-----------------|
| 9 | 月間最大支援金額 | 月別集計の MAX + 該当月 |

| 11 | 緊急事態宣言期間中の支援総額 | 期間指定集計（2020/4/7〜5/25） |

| 14 | 平均支援金額（1回あたり） | 数値カード |
| 15 | 平均支援金額（1店舗あたり） | 数値カード |

### ストーリー指標（定性的）

| # | 指標名 | データソース |
|---|--------|------------|

- **AWS MFA 問題の解決が必須** — DB にアクセスできない限りデータ取得不可
- **AWS 解約（GIGI-6948）前に必ずデータ抽出すること**
- AI&T に DB スキーマ・接続情報を確認できる可能性あり

## Part 3: GitHub Pages サイト構成案

### 技術選定

| 項目 | 選定 | 理由 |
|------|------|------|
| ホスティング | GitHub Pages | 無料、カスタムドメイン対応、期限なし |
| フレームワーク | なし（静的 HTML/CSS/JS） | ビルド不要、メンテナンスフリー |
| CSS | Tailwind CSS (CDN) | モダンなデザイン |
| チャート | Chart.js (CDN) | 軽量、ダッシュボード向き |
| アニメーション | CountUp.js (CDN) | 数値カウントアップ |

### ページ構成（シングルページ）

```
index.html
├── ヘッダー        # さきめしロゴ + ナビ
├── ヒーロー        # メインメッセージ + KPI 4つ（カウントアップ）
├── ストーリー      # さきめしとは + タイムライン
├── ダッシュボード   # 詳細数値 + グラフ
├── メディア        # 掲載一覧（カード形式）+ 受賞歴
├── 動画           # YouTube 埋め込み
├── パートナー      # 自治体/企業連携一覧
└── フッター        # Gigi社情報 + サービス終了案内
```

### ファイル構成

```
sakimeshi-legacy/
├── index.html
├── css/style.css
├── js/main.js          # 数値定数 + カウントアップ + チャート
├── images/
│   ├── sakimeshi-logo.svg
│   ├── og-image.png
│   └── favicon.ico
├── CNAME               # sakimeshi.com
└── 404.html
```

## リスク

| リスク | 影響 | 対策 |
|--------|------|------|
| AWS MFA 解決不可 | DB データ取得不可、DNS 切替不可 | お名前.com でネームサーバー変更して迂回。データは PR TIMES 記載の数値で代替 |
| DB スキーマ不明 | 正確な数値取得不可 | AI&T に確認。Wayback Machine で過去サイト表示数値を確認 |
| メディアリンク切れ | クリック不可 | テキスト表示（媒体名 + タイトル + 日付のみ） |

## STUDIO 案との比較（チーム提案用）

| 項目 | STUDIO 案 | GitHub Pages 案 |
|------|----------|-----------------|
| 費用 | 無料（既存プラン流用） | 無料 |
| 期限 | **2026/12 まで** | **無期限** |
| 内容 | 閉鎖案内テキストのみ | ダッシュボード + メディア + 動画 |
| 価値 | 「閉鎖しました」の告知 | **Gigi の社会貢献の恒久的な記録** |
| 運用 | 藤田が STUDIO で管理 | 毛利が GitHub で管理 |
