# さきめし跡地サイト — デザイン要件

本ドキュメントは mockup.html のデザインに関するユーザー要求を時系列で記録したもの。

---

## 1. 基本方針（初回要求）

> 「懐かしい、エモい、お疲れ様、ありがとう、という感情が想起されるようなデザインにできませんか。」

### 想起すべき感情

| 感情 | ニュアンス |
|------|-----------|
| **懐かしい** | 過去を振り返る、あの頃を思い出す |
| **エモい** | 感傷的、胸にくる、言葉にならない感覚 |
| **お疲れ様** | 労い、役目を終えたものへの敬意 |
| **ありがとう** | 感謝、支援者・飲食店・チームへの謝意 |

→ 「ダッシュボード」や「記録」ではなく、**「思い出のアルバム」「手紙」** のようなトーン。

---

## 2. 視認性（第2回フィードバック）

> 「悪くないですが、ちょっと見づらいですね。」

### 文字

> 「文字は少し大きめ、濃い目に。」

- テキストのフォントサイズを全体的に大きくする
- テキストの色を濃くする（コントラストを上げる）
- 薄い色の文字は読みづらいので避ける

### Hero セクション

> 「Hero 大きすぎます。」

- ヒーローセクションの高さを抑える（`min-h-screen` は大きすぎる）
- 対応: `min-h-[70vh]` に縮小、KPI カードの padding も縮小

---

## 3. アニメーション効果（第2回フィードバック）

### スライドアップ

> 「セクションの上部が表示されると上にすっとスライドするような効果は入れられますか。」

- 各セクションの要素がビューポートに入ったとき、下から上に**すっと**スライドして表示される
- 「すっと」= 急でなく、自然で滑らかな動き
- Intersection Observer で実装

### パララックス窓

> 「背景に画像があり、セクション間の窓が移動して背景画像が見られるような効果もありますよね。」

- セクションとセクションの間に、背景画像が固定されたストリップ（窓）を配置
- スクロールすると窓から背景画像が覗いて見える効果
- `background-attachment: fixed` で実装

---

## 4. カラーパレット（第3回フィードバック）

> 「特設サイトの画像です。色の方向性？は合わせたほうがいいかもしれません。」

参照画像: `https://www.sakimeshi.com/_nuxt/img/b2eb8fb.jpg`
（さきめしキービジュアル: セージグリーン背景、ネイビーのロゴ文字、コーラルピンクのハート、温かみのあるフラットイラスト）

### キービジュアルから抽出した色

| 名前 | HEX | 用途 |
|------|-----|------|
| sage（セージグリーン） | `#8DBEB7` | メイン背景色 |
| navy（ネイビー） | `#2B4C6F` | テキスト・見出し |
| pink（コーラルピンク） | `#E8607A` | アクセント（ハートの色） |
| orange | `#f5a74c` | サブアクセント |
| cream | `#faf7f2` | コンテンツ背景 |
| mint | `#e8f2f0` | コンテンツ背景（交互） |

→ 旧デザインのセピア/ブラウン系から、このセージグリーン + ネイビー + ピンク系に全面変更。

---

## 5. 全体評価と残課題（第4回フィードバック）

> 「かなりいいです。もう少しカードのサイズを調整したりしたら問題ない。」

### カードサイズ調整（未実施）

- 受賞歴カード、メディアカード、パートナーカードのサイズ感を調整する
- 具体的なサイズ指定はなし。現状だと何かしらバランスが気になる程度

---

## 6. ノスタルジー演出の深掘り（第4回フィードバック）

> 「ノスタルジーをイメージさせるような効果は他に何がありますか？」
> 「パララックス窓はその一つだと思ってます。」

- パララックス窓はノスタルジー演出として正しい方向性と認識されている

> 「背景画像がセピアやモノクロ、記憶がおぼろげな雰囲気になってたらなおよし。」

### パララックス窓の背景画像に対する追加要件

| 要件 | 説明 |
|------|------|
| **セピア化** | 背景画像に CSS `sepia()` フィルターをかける |
| **モノクロ化** | 背景画像に CSS `grayscale()` フィルターをかける |
| **記憶がおぼろげ** | ぼかし（`blur()`）や彩度低下（`saturate()`）で朧げな雰囲気を出す |

→ これらは次回改修で実装予定。詳細な演出案は `デザインメモ_ノスタルジー演出.md` に整理済み。

## 8. コンテンツに関する制約

- さきめしロゴ等のアセットは未確保
- サービス終了年月が未確定（「20XX年X月」のまま）
